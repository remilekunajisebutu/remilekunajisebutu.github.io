Sales Analysis Project

Analyzing sales data to uncover trends and improve decision-making

Project Overview

This project focuses on analyzing a retail company's sales data to identify trends, optimize marketing strategies, and forecast future performance. The dataset includes transactional data spanning five years, covering various regions and product categories.

Methodology

- Data cleaning: Removed duplicates and handled missing values.
- Exploratory Data Analysis (EDA): Identified key patterns in sales trends.
- Visualization: Created dashboards to present insights clearly.
- Forecasting: Applied time series analysis to predict future sales.

Tools Used

Python SQL Tableau Pandas Matplotlib

Key Insights

  • Sales peak during the holiday season, with a 40% increase in Q4.
  • Region X contributes the highest revenue, while Region Y shows potential for growth.
  • Product category A has the highest profit margins but requires better inventory management.
